Задача 4
ИНТЕРНЕТ И ИНТРАНЕТ ТЕХНОЛОГИИ / Задачи по работе с файлами

Сделать тестирующую программу. Программа должна состоять из четырёх страниц по три вопроса на каждой. Варианты ответов обозначить переключателями – по три на каждый вопрос. Список номеров правильных ответов хранится в текстовом файле, номера разделены пробелами. На пятой странице программы выводится количество правильных ответов и содержится кнопка “Пройти заново”, которая возвращает пользователя на первую страницу.
